# AlgoTaihua

This is a repository for Taihua's personal algo studies, and a place where she practices writing tests. Fun time!

---

## Installation

Fork, Clone, Install the dependencies for testing

```bash
npm install
```

---

## Run All Tests

```bash
npm test
```
